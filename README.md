CrapBot is a minecraft bot, that is poorly built.

## Installation

1. Clone the repo
```
git clone https://github.com/Janlxrd/minecraftbot.git
```
2. Install all packages needed
```
npm i
```

## Usage

Change the stuff in the `.env` file to your liking.
